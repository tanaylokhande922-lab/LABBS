# **App Name**: CampusConnect

## Core Features:

- User Authentication: Implement secure user sign-up and login using Firebase Authentication (Email/Password and Google Sign-in).
- Path Selection: Provide a 'Choose Your Path' page with buttons for 1st Year, 2nd Year, GATE, and UPSC.
- Resource Display: Display resources (Syllabus, Question Papers, Book PDFs, Subject Treasure) for each selected path.
- File Management: Fetch and display files dynamically from Firebase Storage. PDFs should be viewable in the browser, if possible.
- Data Storage: Use Firestore or Firebase Storage to store and manage all college library resources and data, allowing users to upload if they have permissions.

## Style Guidelines:

- Primary color: A calm, intellectual blue (#468B97) evoking trust and learning.
- Background color: Soft, desaturated light blue (#E1F0F4) to provide a clean and readable backdrop.
- Accent color: A muted, sophisticated green (#64B27D) to add subtle contrast and visual interest.
- Body and headline font: 'Inter', a sans-serif for clear readability and a modern aesthetic.
- Use minimalist, consistent icons for easy navigation (e.g., file icons for documents).
- Mobile-friendly, simple layout using a grid system, potentially implemented using TailwindCSS.
- Subtle transitions and loading animations to improve user experience.
